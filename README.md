# Лабораторная работа №15 “Разработка простого веб-сервиса” (Вариант 3)

## Задание 1
Создан простейший веб-сервис на Flask с маршрутами:
* `/` — главная страница с приветствием
* `/hello/<username>` — персональное приветствие
* `/square/<int:value>` — вычисление квадрата числа

## Задание 2
Реализован маршрут POST `/json`, который принимает JSON с полями:
* `name` — имя пользователя
* `age` — возраст
Сервер возвращает JSON с приветственным сообщением и проверкой корректности данных.

## Задание 3 (Вариант 3)
Реализован маршрут GET `/multiply/<a>/<b>`, который:
* принимает два числа `a` и `b`
* возвращает JSON с исходными числами и их произведением
* проверяет корректность входных данных

## Задание 4
В `main.py` настроен запуск Flask-приложения:
* локальный адрес `127.0.0.1` и порт `5000`
* включён debug-режим для разработки

## Задание 5
Тестирование веб-сервиса:
* GET-запросы проверяются через браузер:
  * `/` — приветствие
  * `/hello/<name>` — персональное приветствие
  * `/square/<number>` — квадрат числа
  * `/multiply/<a>/<b>` — произведение чисел
* POST-запрос с JSON проверяется через Postman или PowerShell:
  * `/json` — обработка данных пользователя и возврат ответа
